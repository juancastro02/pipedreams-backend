# pipedreams-backend

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start

# watch mode
$ npm run start:dev

```

## See the endpoints documentation

```bash
http://localhost:3000/documentation
http://localhost:3000/documentation/json

```
